# HERMES.SH

## Summary
<p>Hermes is a cli-tool that will assist you by providing you links to the documentation 
of the language, framework and configuration files to assist you on your development</p>

----

## Use
<p>Hermes features:</p>
<ul>
    <li>-get_scrolls DIR [DIR ...]</li>
        <p>Creates a compendium of links to documentation helpful based on the 
        project</p>
    <li>-get_scrolls DIR [DIR ...] [-save_scrolls FILE]</li>
        <p>Saves the results of -get_scrolls onto a json type file</p>
    <li>-get_scrolls DIR [DIR ...] [-l ARRAY]</li>
        <p>Returns a specified collection of links [-l ]</p>
    <li>-help</li>
        <p>Provides you with a summary of the commands available and an example</p>
</ul>
